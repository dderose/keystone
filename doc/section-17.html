17.0 :: Popover

The Popover component is a port from Twitter Bootstrap and allows us to display a modeless window on a click event.

Full documentation can be found: http://getbootstrap.com/javascript/#popovers

To use a popover add the class js-popover to an inline element. Block level elements will not work properly as the script won't always be able to accurately align the popover.

The data attribute data-placement can be used to place the popover to the top, left, right, or bottom of the element.

The content for the popover resides inside of a <div> with the class Popover-content. This div must come directly after the element containing the popover call. A heading element can be styled using the Popover-heading class.

To initialize use this script at the end of the <body> tag: 

if ($('.js-popover').length) {
  $('.js-popover').popover({
    container: 'body',
    html: true,
    content: function () {
      return $(this).next('.Popover-content').html();
    }
  });
}

17.1 :: Popover - Light

This modifier changes the appearance of the popover to a light background and border with dark text.

To use this modifier add the following data-attribute to the element: data-modifier="Popover--light".
