21.0 :: Date Pickers

A date picker used to select a single date or a date range.

USAGE:

$('.js-date-picker').daterangepicker({options});

Original docs: https://github.com/dangrossman/bootstrap-daterangepicker

OPTIONS:

The date range picker is configurable by passing options objects and callback functions. A full list of options can be found at https://github.com/dangrossman/bootstrap-daterangepicker#options

Option Description Value Example
Single Date Picker Renders a single calendar to select a single date Boolean $('#js-date-picker').daterangepicker({singleDatePicker:true
});
Opens Whether the picker appears aligned to the left or to the right of the HTML element it's attached to. String('left'/'right') $('#js-date-picker').daterangepicker({opens:'left'
});
Min Date The earliest date a user may select. Date object, moment.js object or string $('#js-date-picker').daterangepicker({opens:'left'
});
Max Date The latest date a user may select. Date object, moment.js object or string $('#js-date-picker').daterangepicker({opens:'left'
});
