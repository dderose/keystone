16.0 :: Tooltip

The Tooltip Tooltip is a port of Twitter Bootsrap's script and allows us to display a short, simple, tooltip on a hover event.

Full documentation can be found: http://getbootstrap.com/javascript/#tooltips

To use a tooltip add the class js-tooltip to an inline element. Block level elements will not work properly as the script won't always be able to accurately align the tooltip.

The data attribute data-placement can be used to place the tooltip to the top, left, right, or bottom of the element.

The content for the tooltip must be entered in the title attribute.

To initialize use this script at the end of the <body> tag: 

if ($('.js-tooltip').length) {
    $('.js-tooltip').tooltip({
      container: 'body'
    });
}
<button class="Button js-tooltip" data-toggle="tooltip" data-placement="left" title="Tooltip on left" data-qe-id="{{tooltip}}">Tooltip Left</button>
<button class="Button js-tooltip" data-toggle="tooltip" title="Tooltip on top" data-qe-id="{{tooltip}}">Tooltip Top</button>
<button class="Button js-tooltip" data-toggle="tooltip" data-placement="bottom" title="Tooltip on bottom" data-qe-id="{{tooltip}}">Tooltip Bottom</button>
<button class="Button js-tooltip" data-toggle="tooltip" data-placement="right" title="Tooltip on right" data-qe-id="{{tooltip}}">Tooltip Right</button>

16.1 :: Tooltip - Light

This modifier changes the appearance of the tooltip to a light background and border with dark text.

To use this modifier add the following data-attribute to the element: data-modifier="Tooltip--light".

<button class="Button js-tooltip" data-toggle="tooltip" data-placement="left" data-modifier="Tooltip--light" title="Tooltip on left" data-qe-id="{{tooltip}}">Tooltip Left</button>
<button class="Button js-tooltip" data-toggle="tooltip" data-modifier="Tooltip--light" title="Tooltip on top" data-qe-id="{{tooltip}}">Tooltip Top</button>
<button class="Button js-tooltip" data-toggle="tooltip" data-placement="bottom" data-modifier="Tooltip--light" title="Tooltip on bottom" data-qe-id="{{tooltip}}">Tooltip Bottom</button>
<button class="Button js-tooltip" data-toggle="tooltip" data-placement="right" data-modifier="Tooltip--light" title="Tooltip on right" data-qe-id="{{tooltip}}">Tooltip Right</button>
